## Pre-render checks

- [ ] `check_text_width.py` прогнан — 0 overflow warnings
- [ ] Все шоты длятся 2-7с (нет «застывших» >8с без обоснования)

## QC после render

- [ ] Длина = длине аудио
- [ ] Sidechain ducking работает (слышно как музыка приглушается под голосом)
- [ ] Music swell на 2 key moments (+6dB)
- [ ] Teal-orange color grade применён
- [ ] Текст в safe zone (top ≥180, bottom ≤1620, sidebar fits)
- [ ] Title-визуал совпадение проверено в финальном просмотре
- [ ] Размер итогового MP4 в рабочих пределах (≤30MB для inline-доставки)
