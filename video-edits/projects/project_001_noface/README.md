# Project 001 — «Обложка паспорта из Таиланда»

**Тип:** «Рассказ без лица» (no-face Reels)
**Формат:** Вертикаль 1080x1920
**Длительность:** = длине аудио = **53.5 секунды**

## Концепция

Travel/lifestyle Reels о персональной обложке для паспорта, купленной в Таиланде. От первого «бликующего» кадра паспорта до финальной мысли «ваш паспорт — ваша история».

## Применимые техники из библиотеки

Учитывая концепцию (lifestyle/travel/тактильность):

- **B-roll-метафора** в каждой секции (refS s124 принцип — visual rhyme к message)
- **Multi-tier title на финале** (Dela Gothic + Caveat пара) для CTA-карточки «ВАШ ПАСПОРТ — ВАША ИСТОРИЯ»
- **iOS highlighter** (refY v4) на ключевых словах в субтитрах
- **Цена крупно** как display-элемент (refU u13 «hook-with-money» приём) для секции 35-48с
- **Top-down кадр** (refM m4 / реф 7 Wes Anderson) если есть подходящий B-roll процесса

## Подбор SFX (под движение текста + кадра)

- **При появлении каждого титра** (5 раз): `whoosh_grainy_*` или `dust_swoosh`
- **При смене сцены** (4 перехода): `proletai_transition` или `low_transition_6group`
- **На моменте «120 бат»** (акцент-цена): `cartoon_bubble_pop` или `puff_smoke_1`
- **На финальной карточке**: `film_title_transition`
