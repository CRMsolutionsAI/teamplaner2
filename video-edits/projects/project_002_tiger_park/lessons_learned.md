# Project_002 tiger_park — Lessons Learned

**Финальная версия:** `v_final.mp4` v5 (108.2с, vertical 1080x1920)
**Дата:** 2026-05-19

## ✅ Что получилось лучше всего

1. **Window-in-black парадигма** — заказчица сказала «выглядит дорого». Видео-окошко (720x1200 CENTER / 600x1000 LEFT/RIGHT) на чёрном canvas + текст рядом, не поверх. Перенесено в CLAUDE.md как дефолт для cinematic-mood.

2. **«Польза-карточка» на climax** — «НЕ ЩЕКОЧИ СИЛУ» + подпись «правило работает: переговоры • команда • отношения». Превратило «личную историю про тигра» в applicable life-правило. Reusable шаблон для любого обучающего видео.

3. **Slow-mo на ключевом моменте** — тигр в бассейне 0.5x под текст «земля-инь и ПЛАМЯ» = мистическая встреча стихий. Один шот тащит весь сегмент M7.

4. **Hook-визуал крупным планом** — переключение с POV-руки (v1-v4) на крупный план тигриного торса (v5) = разница между «pass» и «смотрит дальше». Зашёл в CLAUDE.md как обязательное правило.

## 🎯 Конкретное усложнение для следующего ролика

**Реальная анимация text-reveal** — сейчас fade-in 250мс работает, но slide-in (текст влетает сбоку с motion blur) выглядит ещё богаче. Попробовать на следующем проекте: `drawtext` + время-зависимое смещение x через `x='if(lt(t,t0+0.25), x_end + 300*(1-(t-t0)/0.25), x_end)'`.
