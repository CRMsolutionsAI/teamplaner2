# Кириллические шрифт-пары — каталог от @marrssiankaa

Профессиональные шрифт-пары специально для **кириллицы**. Каждая пара = display-шрифт + body-шрифт.

## Каталог пар

| # | Display | Body | Стиль | Когда применять |
|---|---|---|---|---|
| 1 | **Druk Cyr** | Iiana (handwriting script) | Брутальный + рукописный | Динамика, energy, спортивные/мотивационные ролики |
| 2 | **Dela Gothic** | Denistina (script) | Объёмный bubble + handwriting | Friendly tutorials, fun-контент |
| 3 | **LS Chalk** | Gilroy (geometric sans) | Меловой + гротеск | Hand-drawn эстетика, обучающие ролики |
| 4 | **Argent CF** | PF DinDisplay | Modern serif + display sans | Premium, элегантность, дорогие продукты |

## Где брать шрифты

- **Druk Cyr** — Commercial Type (платный). Альтернатива: Inter Black, Heading Now Black, Bebas Neue Cyr
- **Dela Gothic** — Google Fonts (бесплатно, есть кириллица: Dela Gothic One)
- **LS Chalk** — пакет от @marrssiankaa («комментарий КАНАЛ»)
- **Argent CF** — Connary Fagen Type Design
- **PF DinDisplay** — ParaType
- **Gilroy** — Radomir Tinkov (free for personal)

## Бренд-рекомендации для Натальи

**Текущий стек:** Roboto Condensed Bold + DejaVu Sans + Caveat

**Возможные апгрейды:**

| Сценарий | Пара | Что даёт |
|---|---|---|
| Энергичный промо (Wispr-стиль) | **Dela Gothic + Caveat** | Friendly + уже знакомый handwriting |
| Премиум-продукт | **Argent CF + PF DinDisplay** | Элегантность для дорогих курсов |
| Образовательный туториал | **LS Chalk + Gilroy** | «Я тебе расскажу как друг» |
| Динамичный мотивационный | **Druk Cyr + Caveat** | Брутальный display + дневниковый body |

## Принцип шрифт-пары

**Контраст:** display массивный → body тонкий/handwriting.
**Иерархия:** display = 80-200px, body = 24-48px.
**Кириллица:** обязательно проверять что шрифт поддерживает кириллицу (не все — Druk имеет только Cyr-версию платно).
