## Техники оформления субтитров (5 туториалов)

### 1. Размытая подложка
Под текстом субтитра — мягкий полупрозрачный размытый прямоугольник (`boxblur` или drawbox с opacity ~0.4). Текст читается на любом фоне, не отвлекает.

### 2. Подвижная подложка
Прямоугольник-подложка под текстом сдвигается/растягивается с появлением слова (анимация). Текст «вытесняет» подложку. Можно через `drawbox` с временно меняющимися w/x.

### 3. Подложка из стикеров (фото-коллаж)
Под текстом — текстурный коллаж из маленьких фото/стикеров. Эстетичный визуал-фон, особенно для travel/lifestyle. В ffmpeg сложно — нужна заготовка PNG-картинка.

### 4. IOS выделение текста
Yellow/pink/orange полоска-маркер ПОЗАДИ конкретных слов (как highlighter в Apple Notes). Реализация через drawbox с `t=fill` и `color=0xFFEB3B@0.7` под `drawtext`. **Применимо к нашему стилю** — оранжевая подсветка `0xFF5722@0.7` под ключевыми словами вместо просто оранжевого цвета шрифта.

### 5. Тренды в оформлении сабов в Reels (общий обзор)
- Сабы не статичные — они «дышат», двигаются вместе со словами
- Эмодзи/иконки рядом для эмоции
- Уникальный шрифт = бренд
- Цвет акцента = бренд (как наш оранжевый)

### Что применять в следующих видео
| Техника | Когда |
|---|---|
| Размытая подложка | Светлый/пёстрый фон демо, чтобы белый текст читался |
| Подвижная подложка | Каждое слово появляется с растягиванием полоски |
| IOS highlighter | Подсветить КЛЮЧЕВОЕ слово внутри фразы — оранжевая полоса позади |

## Ещё 4 техники сабов (от @o.siasia, продолжение)

### 6. Чёрная тонкая обводка
Текст с **тонкой** чёрной обводкой (Outline=1-2 в ASS, или drawtext с `borderw=1-2`). Чище и аккуратнее жирной обводки. Хорошо на смешанных фонах.

### 7. Съехавшая подложка (offset backdrop)
Прямоугольник-подложка смещён на 4-8px по диагонали от текста — создаёт «3D-тень». Реализация: два слоя drawbox+drawtext со смещением координат.

### 8. Пиксельная подложка
8-битный/ретро стиль фона под текстом — пикселизация. Можно через `pixelize` filter или заготовленную PNG-текстуру.

### 9. Размытые геометрические фигуры
Большие мягкие круги/овалы пастельных цветов как декорация на фоне (не под текстом, а в стороне). Композиционные элементы. SVG → PNG → overlay.

## refP_motivational_graffiti/ — Серия эмоциональных Reels (готовый продукт)

**Тип:** Готовый продукт (motivational/inspirational mini-Reels)
**Стиль:** Brutal-emotional, кистевой шрифт + монохром + freeze-frame
**Контент:** Мотивация, страх, время, сомнения

### Быстрый анализ (этап 1)

**Тип хука:** Шок-фраза в первой секунде. Первое слово гигантское. Хук = вся фраза за 2-5с.
**Длительность роликов:** 2-5с — мини-формат, одна мысль = один кадр.
**Шрифт:** Кистевой graffiti, неровный штрих, имитация маркера. ОДИН шрифт = голос бренда.
**Цвет:** монохром текста — жёлтый ИЛИ красный, никогда не белый. Один акцент на всю серию.
**Размер:** 30-50% высоты кадра — слово как объект.
**Расположение:** центр (p6/p8) или низ (p10), word-by-word reveal.
**Темпоритм:** очень медленный — 2-3с на слово.
**Переходы:** match-cut по позиции, freeze-frame + word reveal.
**Цветокор:** монохром/дуотон + один акцент-цвет, высокий контраст, силуэт-glow.

## refM_text_effects/ — Серия premium-туториалов по text effects

**Автор:** Неизвестен, англо/немецкоязычный design-tutorial аккаунт.
**Стиль:** Premium production, минималистичный, дизайнерский.
**Фишка серии:** Каждое видео = 1 текстовая техника, реализуемая в Premiere Pro / After Effects.

### m1 — «creative TEXT» × paintbrush (3.4с)
**Техника:** Text-object physical interaction
- Кисть капает красной краской на синий фон
- Текст «creative» (serif italic) появляется маленьким → big bold «TEXT» под ним
- Бирюзовый чекмарк (✓) акцентом
- Краска визуально «обтекает» текст
- **Шрифты:** serif italic (decorative) + bold sans (display) — премиум-пара
- **Цвет:** красный + синий + бирюзовый акцент

### m2 — «BACK GROUND» × z-layering (4.0с)
**Техника:** Person passes BEHIND massive text
- Человек идёт к скамейке у океана
- БОЛЬШИЕ жирные буквы «BACK GROUND» (стек по 2 строки)
- Цвет анимируется: розовый → красный
- Человек проходит ЗА текстом (text in foreground)
- **Шрифт:** ультра-bold sans-serif (Helvetica Black / Druk)
- **Цвет:** magenta → crimson по ходу видео

### m3 — «stack den TEXT wie ein Pro» × multilayer (2.3с)
**Техника:** 3-tier multi-size composition
- Парень на жёлтом стуле в парке
- 3 уровня текста в одной композиции:
  - handwriting script красным «stack den» (decorative)
  - **БОЛЬШОЕ** жирное чёрное «TEXT» (display)
  - мелкое красное «wie ein Pro» (caption)
- Иерархия: decorative → display → caption
- **Язык:** немецкий
- **Шрифты:** handwriting (Caveat-like) + ultra-bold sans + thin sans

### m4 — «OUTLINE» × masking через объект (3.4с)
**Техника:** Text wraps around subject via masking
- Top-down кадр (как ref7 «Wes Anderson») — человек с красным MacBook
- ОГРОМНОЕ красное «OUTLINE» поперёк кадра
- Часть текста ЗА человеком, часть ПЕРЕД — masking создаёт «прохождение через текст»
- Чистый белый фон делает эффект ярким
- **Шрифт:** display ultra-bold (Druk Wide / Roboto Condensed Black)
- **Композиция:** top-down + минимализм + один акцентный цвет

### m5 — «blend mode DIFFERENCE» × цвето-инверсия (5.1с)
**Техника:** Premiere blend-mode «Difference» для интеграции текста с фоном
- Скриншот таймлайна Premiere Pro в нижней трети
- Превью: кисть+красная краска под текстом
- Текст «blend mode DIFFERENCE» (serif italic + bold sans)
- Demo переключения Blend Mode: Normal → Darken → Difference
- При «Difference» цвета инвертируются RGB-фильтром (красный → циан)
- **Техническая инструкция:** Effect Controls → Opacity → Blend Mode → Difference
